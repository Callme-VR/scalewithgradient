# ScaleWithLine

A Next.js landing page component featuring decorative scale lines and gradient styling. Built with Tailwind CSS v4 and shadcn/ui.

## Features

- **Decorative Scale Lines** - Horizontal and vertical measurement-style borders using CSS gradients
- **Gradient Styling** - Blue gradient buttons and text effects
- **Responsive Layout** - Full viewport height design with centered content
- **Navigation & Hero** - Clean navbar with CTA buttons and hero section

## Tech Stack

- [Next.js](https://nextjs.org) 16.2.4
- [React](https://react.dev) 19.2.4
- [Tailwind CSS](https://tailwindcss.com) v4
- [shadcn/ui](https://ui.shadcn.com) components
- [Lucide React](https://lucide.dev) icons

## Getting Started

Run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) to view the result.

## Project Structure

- `components/scalewithline.tsx` - Main landing page component with scale lines
- `components/ui/button.tsx` - shadcn/ui Button component
- `app/page.tsx` - Page entry point

## Deployment

Deploy on [Vercel](https://vercel.com/new) or any Next.js-compatible platform.
